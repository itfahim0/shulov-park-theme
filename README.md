A custom-built, highly responsive WooCommerce theme designed specifically for **Shulov Park**, a premium online grocery and organic food store in Bangladesh. Inspired by modern e-commerce trends (such as Ghorer Bazar), this theme focuses on a clean user interface, fast checkout, and localized payment integrations.

## ✨ Features

- **Custom Brand Aesthetics:** Beautiful green (`#388e3c`) and orange (`#ff9800`) color palette designed for organic/fresh food stores.
- **Dynamic Homepage:** Fully compatible with the WordPress Block Editor (Gutenberg). Add product grids, hero banners, and text rows directly from the WP Admin panel.
- **Quick "Buy Now" Checkout:** Custom logic that allows users to bypass the cart and go straight to the checkout page for faster conversions.
- **Responsive Header & Navigation:** Clean top-bar featuring a product search bar, interactive cart icon with real-time count, and user account shortcuts.
- **Customizer Options:** Easily edit the Support Phone Number and "Accepted Payments" text directly from `Appearance > Customize > Footer Options`.
- **Localized Payment Footer:** A colorful, hard-coded grid of popular Bangladeshi payment gateways (bKash, Nagad, Rocket, Visa, Mastercard) to build customer trust.
- **Mobile First:** Fully optimized for mobile devices with stacked grids, responsive menus, and touch-friendly buttons.

## 🚀 Installation

1. Download the theme as a ZIP file (or clone this repository).
2. Compress the `shulov-park` folder into `shulov-park.zip`.
3. Log in to your WordPress Admin Panel (`wp-admin`).
4. Navigate to **Appearance > Themes**.
5. Click **Add New** and then **Upload Theme**.
6. Upload the `shulov-park.zip` file and click **Install Now**.
7. Once installed, click **Activate**.

## ⚙️ Requirements

- WordPress 5.0 or higher
- **WooCommerce** plugin installed and activated
- PHP 7.4 or higher

## 🛒 WooCommerce Setup

After activating the theme, ensure you configure the following in WooCommerce:
1. Go to **WooCommerce > Settings > Advanced**.
2. Assign the **Cart Page** and **Checkout Page**.
3. To display products on the homepage, edit your Home page and insert the `[products limit="8"]` shortcode.
